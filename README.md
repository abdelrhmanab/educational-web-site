# 📚 EduMaster – الموقع التعليمي الحديث

![EduMaster Preview](https://via.placeholder.com/1200x400?text=EduMaster+Preview)

**EduMaster** هو موقع تعليمي متطور يقدم كورسات عالية الجودة للمستخدمين الراغبين في تطوير مهاراتهم في مختلف المجالات. الموقع مصمم بشكل حديث وجذاب مع تجربة مستخدم سلسة، باستخدام أحدث تقنيات الويب مثل **React**, **Tailwind CSS**, **DaisyUI**, و **React Router**.  

---

## ✨ ميزات الموقع

- **واجهة مستخدم جميلة وحديثة**
  - تصميم متجاوب تمامًا مع جميع الأجهزة (Desktop / Tablet / Mobile)
  - عناصر UI متطورة باستخدام **DaisyUI** و **React Icons**
- **التنقل السلس**
  - Navigation ديناميكي باستخدام **React Router**
  - Dropdown menus ذكية و **Accordion** في قائمة الموبايل
- **الكورسات**
  - عرض الكورسات مع صفحات تفصيلية لكل كورس
  - دعم القوائم المتعددة وتنقل سلس داخل الكورس
- **الصفحات الأساسية**
  - Home, About, Courses, Blog, Profile, Login, Signup, Contact
- **نظام تصميم متقدم**
  - استخدام **Tailwind CSS** بشكل احترافي
  - ألوان متناسقة، تباين ممتاز، وأزرار تفاعلية
- **التجربة التفاعلية**
  - Search bar ذكي للكورسات
  - Dropdown و Sidebar متجاوب
  - Active links توضح الصفحة الحالية باستخدام **NavLink**
- **Responsive Design**
  - Drawer Sidebar على الموبايل
  - Navbar ديناميكي يتكيف مع جميع أحجام الشاشات
- **قابل للتوسعة**
  - جاهز لإضافة ميزات متقدمة مثل الدورات المدفوعة، التعليقات، أو ملفات الفيديو

---

## 🛠️ التقنيات المستخدمة

| التقنية | الدور |
|---------|-------|
| **React** | لبناء واجهة المستخدم التفاعلية |
| **React Router** | إدارة الصفحات والتنقل الديناميكي |
| **Tailwind CSS** | تصميم حديث وسريع الاستجابة |
| **DaisyUI** | مكونات UI جاهزة وحديثة |
| **React Icons** | أيقونات جذابة وسهلة الاستخدام |
| **TypeScript (اختياري)** | كتابة أكواد آمنة ومنظمة |
